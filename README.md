# IotLoggerClientSample

## Overview

This program is a sample to send data to IotLogger.
Send the signal received by the serial port to the server.

## Dependency
* python 3.8.3

## How to set up

1. install python dependancy module

    ```
    $ pipenv install
    ```

## Usage

start 

```
$ ./start.sh
```
